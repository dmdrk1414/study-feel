#### p. 8 빅데이터란 무엇인가

- 정의 

  1. 서버 한 대로 처리할 수 없는 규모의 데이터

  2. 기존의 소프트웨어로는 처리 할 수 없는 규모의 데이터

  3. 3V (**Volume: 크기, Velocity: 생성속도, Variety: 다양성** ) + 2V (**Value: 가치, Varacity: 정확성**)

     sns 데이터 = 3V ,  + 가치있는 데이터 = 2V

<img src="C:\Users\Park-Seung-Chan\AppData\Roaming\Typora\typora-user-images\image-20221016000944324.png" alt="image-20221016000944324" style="zoom:67%;" />

- 서버 업그레이드 방법
  1. Scale Up
     - 한대를 크게 업그레이드한다.
  2. Scale Out
     - 서버를 병렬방식으로 여러대를 붙인다.

#### p. 11 빅데이터 및 인공지능 구현 과정

![image-20221016001419324](C:\Users\Park-Seung-Chan\AppData\Roaming\Typora\typora-user-images\image-20221016001419324.png)

1. 데이터 수집
   - 단순 데이터 수집
   - 빅데이터 플랫폼
   - 데이터베이스
   - Open API 활용
   - 웹 크롤링
2. 데이터 저장
   - 일반 파일
3. 데이터 처리
   - 필터링
   - 이상치, 결측치 처리 : 이상한 데이터, 없는 데이터 검사

   - 데이터 구조 파악 : 몇건의 데이터가 좋은가 특정 분석

   - 변수 유형 확인
   - 상관 분석

   - 변수 선택 : String, int 인지 확인
   - 특징 추출
   - 차원, 축소 : 범위 값을 맞춘다.
   - 정규화 : 키제기 라고 한다. (20 - 30 을 1 - 10으로 변경)
   - 표준화
6. 데이터 구성
   - 홀드 아웃 : 데이터를 8 대 3으로 나눈다
   - 교차 검증 : 검증을 따로한다.
   - 부트스트랩
7. 데이터 분석 또는 인공지능 모델
   - 데이터 분석
     1. 통계적 분석
     2. 머신러닝
     3. 딥러닝
   - 인공지능 모델
     1. 모델 생성 : 뇌구조 생성
     2. 모델 설정 : 얼마가 학습인가 (학습 - 검증)
     3. 모델 학습 (Train)
     4. 모델 검증(Validation) : 평가를 하는 과정
     5. 모델 튜닝
     6. 모델 평가(Test)
8. 데이터 시각화 및 서비스
   - 데이터 시각화
     1. 그래프 : 막대
     2. 인포 그래픽 : 그림
   - 서비스
     1. Web 서비스
     2. App 서비스
