#include <iostream>
#include <stack>

using namespace std;

int main(void) {
	stack<int> s;
	s.push(7);
	s.push(5);
	s.push(4);
	s.pop();
	s.push(6);
	s.pop();
	while(!s.empty()) {
		cout << s.top() << ' ';
		s.pop();
	}
	return 0;
}
스택이란 상하차의 개념이다. 먼저들어간것이 먼저 나가는 구조이다.
push(element): top에 원소를 추가
pop(): top에 있는 원소를 삭제

조회
top(): top(스택의 처음이 아닌 가장 끝)에 있는 원소를 반환
기타
empty(): 스택이 비어있으면 true 아니면 false를 반환
size(): 스택 사이즈를 반환
